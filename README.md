# VisREcall - Visual Reference Search

A text-to-image retrieval tool designed for artists. It helps you search your saved visual references, on platforms like Pinterest or in local folders, by simply describing them in natural language. No need for tags, folders, or perfect memory. 

The search application uses CLIP (Contrastive Language-Image Pre-training) to find images based on text descriptions or similar images. Built with Streamlit and Python.

## Features

- 🔍 Search images using text descriptions
- 🖼️ Search using similar images
- 📊 Real-time similarity scoring
- 🚀 Fast and efficient search using CLIP embeddings
- 💾 Persistent storage using ChromaDB
